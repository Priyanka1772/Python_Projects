## Diwali Sales Analysis using Python (Exploratory Data Analysis)

## Project Overview
* This project performs exploratory data analysis on Diwali sales data to understand customer purchasing behavior and sales trends.
* Python was used to clean, analyze, and visualize raw transactional data.
* The analysis demonstrates how Python can be effectively used for business-focused data analysis.

## Features
* Data Cleaning & Preparation: Handling missing values and data type corrections.
* Exploratory Data Analysis (EDA): Analysis of customer demographics, product categories, and regional sales patterns.
* Visualization: Clear visual representations of trends and distributions.
* Insight Generation: Business-driven insights derived from data exploration.

## Technologies Used
* Python
* pandas
* numpy
* matplotlib
* seaborn

## Key Skills Demonstrated
* Data Cleaning & Transformation
* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Insight Generation
* Analytical Thinking with Python

## How To Access
* The Jupyter Notebook (.ipynb) file has been uploaded to this repository.
* Open the notebook using Jupyter Notebook or JupyterLab.
* Run the cells sequentially to reproduce the analysis and insights.
